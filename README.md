# HireFlow — Speech → HR Requirement Extractor

## 🚀 What is this?

HireFlow is a simple AI-powered tool that converts hiring conversations into structured HR requirements.

You can:

* Speak or paste a conversation
* Extract key hiring details
* Generate ready-to-use outputs (JD, email, etc.)

---

## 🎯 Why this exists?

HR teams spend hours:

* Taking calls
* Writing notes
* Converting them into structured job descriptions

HireFlow reduces that effort to seconds.

---

## ⚡ How it works

1. Capture conversation (voice / text)
2. AI extracts key hiring fields
3. Review & select fields
4. Generate structured outputs

---

## 🔐 Privacy & Data Safety (Important)

Your data is **NOT stored anywhere**.

* ✅ Everything runs in your browser
* ✅ No database storage
* ✅ No conversation is saved
* ✅ No personal data is retained

👉 The app only **processes input temporarily** to generate output
👉 Once you refresh or leave, data is gone

You can use this tool **without worrying about data leakage**

---

## 💡 Key Features

* 🎙 Voice & text input
* 🧠 AI-based HR field extraction
* 📄 Structured output (fields, JD, email)
* ➕ Custom field support
* 🌐 Hinglish / English support
* ⚡ Fast & lightweight
